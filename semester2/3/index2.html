Unordered HTML

Unodered HTML adalah jenis tampilan daftar yang menggunakan tanda bulat
atau lingkaran untuk setiap elemen, Biasanya digunakan untuk membuat daftar
yang tidak memiliki urutan atau prioritas tertentu.

Ordered HTML

Ordered HTML adalah jenis tampilan daftar yg menggunakan nomor atau angka untuk setiap elemen
Biasanya digunakan untuk membuat daftar yg memiliki urutan atau prioritas tertentu.

Definition List HTML

Definition List pada HTML adalah jenis tag yg digunakan untuk membuat daftar definisi dengan menggunakan tag DL untuk membungkus daftar tersebut,tag DT untuk menandai judul
atau istilah,dan tag DD untuk menandai definisi dari masing-masing istilah tersebut...
