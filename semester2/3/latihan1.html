kenapa kuliah IT itu penting?

di perkembangan zaman skrng sudah memakai teknologi canggih dimana itu berhubugan dengan IT,
jadi kuliah IT itu penting karena:

  • banyak perusahaan yg sendang mencari karyawan di bidang IT
  • IT berhubugan dengan perkembangna zaman skrng
  • pelajaran yg diajarkan memiliki peluang kerja yg tinggi
  • bisa memahami lebih dalam tentang teknologi
  • bisa membantu orang yg kurang mengetahui tentang perkembangan tekmnologi

langkah-langkah menjadi cyber security system

Cyber security System adalah pekerjaan yg dimana kita menjaga suatu sistem
dari para penyerang /hacker yg ingin mengambil sebuah data dari sistem tersebut.
inilah langkah-langkah menjadi Cyber Security System:

  1. mempunyai perangkat yg memadai
  2. mempunyai semua syarat2 menjadi Cyber Security
  3. mempunyai certifikat cyber security
